# Buscador de CEP

Um simples buscador de CEP que permite aos usuários encontrar informações de endereços no Brasil.

## Funcionalidades

- Insira um CEP válido para encontrar informações detalhadas sobre o endereço.
- Modo de visualização noturno para uma experiência de usuário mais confortável em ambientes com pouca luz.

## Como Usar

1. Insira o CEP desejado no campo de busca.
2. Clique no botão "Buscar".
3. Visualize os resultados do endereço.


## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript
- jQuery
- [Font Awesome](https://fontawesome.com/)

## Instalação

1. Clone este repositório.
2. Abra o arquivo `index.html` em seu navegador.

## Estilos

- O aplicativo possui um estilo moderno e responsivo.
- Suporta um modo de visualização noturno para reduzir a fadiga ocular em ambientes escuros.
